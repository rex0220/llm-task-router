# 記事テーマ: 生成AIを日常的に使うようになって変わったこと

エンジニアが生成AI（ChatGPTやClaudeなど）を日々の仕事に取り入れて感じた、考え方や働き方の変化を綴る読み物。

## 対象読者

- これから生成AIを使ってみたい、または使い始めたばかりの人
- 技術者でなくても読める一般読者

## 記事のゴール

- 生成AIを「魔法の箱」ではなく「相棒」として捉える感覚が伝わる
- 任せること／自分で判断することの線引きの考え方が伝わる
- 読後に「自分も少し試してみよう」と思える

## 含めたい論点

- 最初に感じた期待と、実際に使ってみたギャップ
- うまくいった使い方（壁打ち・下書き・要約など）の具体例
- 任せてはいけないと感じた場面（事実確認・最終判断）
- AIに頼ることへの不安と、それとどう折り合いをつけたか
